# Process Overview

- Connect to IRC server
- Join IRC channel
- Get search terms from user
- Send search terms into channel
- Receive and process DCC SEND
- Send DCC GET
- Write received file to disk
- Unzip search results
- Get search results from file
- Display search results to user
- User selects file from search results
- Send filename with bot prefix into channel
- Receive and process DCC SEND
- Send DCC GET
- Write received file to disk
- Unzip downloaded archive
- Optionally invoke kepubify on the ebook
